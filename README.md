# PICO-CUBE
Pico Cube is a 4x4x4 LED cube HAT for Raspberry Pi Pico with 5V DC operating voltage. Pico cube, a monochromatic( Red, Blue, Green) 64 LED, is a fun way to learn programming. It is designed to perform incandescent operations with low energy consumptions, robust outlook, and easy installation that make people/kids/users learn the effects of LED lights with a different pattern of colours via the combination of software and hardware i.e. Raspberry Pi Pico.
